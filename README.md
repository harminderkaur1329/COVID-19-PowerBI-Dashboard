# 🦠 COVID-19 Dashboard using Microsoft Power BI

## 📌 Project Overview

This project is an interactive COVID-19 analytics dashboard developed using Microsoft Power BI. It provides an executive summary and detailed analysis of global COVID-19 data, enabling users to explore confirmed cases, deaths, recoveries, and regional trends through interactive visualizations.

---

## 🎯 Objectives

- Analyze worldwide COVID-19 trends
- Compare WHO regions and countries
- Monitor confirmed cases, deaths, and recoveries
- Explore recovery and death rates
- Provide interactive filtering using slicers

---

## 🛠️ Tools & Technologies

- Microsoft Power BI
- Power Query
- DAX
- Excel Dataset

---

## 📈 Key Insights

- Americas recorded the highest number of confirmed COVID-19 cases.
- Recovery rate exceeded **57%**.
- Death rate remained below **4%**.
- Interactive slicers allow users to filter data by country and WHO region.
